\# AI Prompt Logs \& Workflows

In strict compliance with the BUS-629 AI Use Policy, this directory stores comprehensive logs of all interactions with Large Language Models (LLMs) like ChatGPT and Claude.



\## Purpose

To maintain academic transparency and showcase advanced AI augmentation skills by capturing system prompts, iterative refinements, and verification steps.



\## Naming Convention

Logs must be documented as: `YYYY-MM-DD-{analytical-stage}-prompt-log.md`.
