\# Documentation Directory

This directory serves as the central repository for all written deliverables, qualitative memos, and technical specifications produced during the BUS-629 course.



\## Subdirectory Structure

\* `/decisions`: Analytical memos and strategic decision-making documents.

\* `/specs`: Technical specifications and prompt guidelines for generative AI tools.

\* `/plans`: Project timelines, milestones, and strategic planning artifacts.

\* `/templates`: Stubs and baseline references pointing to canonical course templates.
